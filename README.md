# 🎮 Maze Runner

A fun and interactive maze escape game built with HTML5 Canvas and vanilla JavaScript!

## Features

✨ **Dynamic Maze Generation** - Each game generates a unique maze using recursive backtracking algorithm

🎯 **Difficulty Levels** - Easy, Medium, and Hard modes

⏱️ **Time Tracking** - Track your completion time with best time scoring

💡 **Hint System** - Get visual guidance showing the path to the exit

⏸️ **Pause Feature** - Pause and resume your game at any time

📱 **Responsive Design** - Works on desktop and mobile devices

## How to Play

1. **Goal**: Guide the blue square through the maze to reach the green exit
2. **Controls**: 
   - Use arrow keys (↑ ↓ ← →) or WASD to move
   - Press **P** to pause/resume
   - Click buttons for New Game, Pause, and Hint
3. **Tips**: Avoid walls, minimize moves, and beat your best time!

## Getting Started

Simply open `index.html` in your web browser and start playing!

## Files

- `index.html` - Game interface and styling
- `game.js` - Game logic and mechanics
- `README.md` - This file

## Technical Details

- **Maze Generation**: Uses recursive backtracking algorithm for perfect maze generation
- **Pathfinding**: BFS (Breadth-First Search) algorithm for hint system
- **Graphics**: HTML5 Canvas 2D context
- **Storage**: localStorage for best time persistence

## Scoring

- **Time**: Complete the maze as fast as possible
- **Moves**: Track your movement count
- **Best Time**: Your best completion time is saved locally
